**Установка**

```
git clone https://github.com/wowshakhov/apa.git
```

**Использование**

```
apsum(x, y)
```
- возвращает сумму двух чисел x + y

```
apsub(x, y)
```
- возвращает разность двух чисел x - y

```
apmul(x, y)
```
- возвращает произведение двух чисел x * y

Аргументы предоставляются в виде строк. Допускаются отрицательные числа.

**Пример**

```
apsum("3", "-2");
```
